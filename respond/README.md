# RESPOND (RS) - Secondary Category

The Respond function includes appropriate activities to take action regarding a detected cybersecurity incident.

## Subcategories

### RS.MA - Incident Management
Automated incident response, purple teaming, collaborative threat response, incident handling workflows.

### RS.AN - Incident Analysis
Forensic analysis, root cause analysis, threat intelligence correlation, post-incident investigation.

### RS.CO - Incident Response Reporting and Communication
Stakeholder notifications, regulatory reporting, public communications, incident disclosure.

### RS.MI - Incident Mitigation
Containment strategies, eradication procedures, system isolation, threat neutralization.
